# SDSS2507 3Ni α/γ Orientation-Relationship EBSD

MATLAB/MTEX workflow and EBSD data for α/γ orientation-relationship analysis in LPBF-fabricated 3 wt.% Ni-modified super duplex stainless steel 2507.

## Overview

This repository contains MATLAB/MTEX scripts and supporting EBSD-based outputs for analyzing the crystallographic orientation relationship between ferrite (α) and austenite (γ) in LPBF-fabricated 3 wt.% Ni-modified SDSS 2507.
